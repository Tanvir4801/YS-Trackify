YS Trackify – Labour Management App

YS Trackify is a Flutter-based mobile application designed to simplify labour attendance and payment tracking for construction sites and small businesses.

⸻

📱 Features
	•	✅ Labour Attendance Tracking
	•	💰 Payment History Management
	•	📅 Monthly Attendance Filter
	•	🔐 Supervisor PIN Security
	•	📊 Clean and User-Friendly Dashboard

⸻

🛠️ Tech Stack
	•	Flutter (UI Development)
	•	Dart (Programming Language)
	•	Hive (Local Database)

⸻
🎯 Use Case

This app is useful for:
	•	Construction site supervisors
	•	Small contractors
	•	Workforce management

⸻

📌 Future Improvements
	•	🔄 Cloud Sync (Firebase)
	•	📍 Location-based attendance
	•	📊 Advanced analytics dashboard
